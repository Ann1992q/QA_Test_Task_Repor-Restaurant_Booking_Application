Для тестирования сайта "InfoSphere" необходимо создать три основных документа: тест-план , чек-лист и тест-кейсы . 

I Тест-план

Цель тестирования: Проверка функциональности, удобства использования, производительности и безопасности сайта "InfoSphere" с целью обеспечения бесперебойной работы, минимизации ошибок, достижения времени загрузки страниц менее 3 секунд и повышения удовлетворенности пользователей.
Область применения: Документ используется тестировщиками для планирования и выполнения тестирования, разработчиками для исправления дефектов и менеджерами для оценки качества продукта.

1. Объем тестирования
Функциональное тестирование : Проверка работы всех ссылок, форм, кнопок, меню и других элементов интерфейса.
UI/UX тестирование : Оценка удобства использования, дизайна и навигации.
Кроссбраузерное тестирование : Проверка совместимости с браузерами (Chrome, Firefox, Edge, Safari).
Тестирование производительности : Измерение времени загрузки страниц, отзывчивости сервера.
Тестирование безопасности : Проверка на наличие уязвимостей, таких как XSS, SQL-инъекции, CSRF.
Мобильное тестирование : Проверка отображения и функциональности на мобильных устройствах.

2. Подход к тестированию
Ручное тестирование : Для проверки UI/UX, кроссбраузерной совместимости.
Автоматизированное тестирование : Для регрессионного тестирования и тестирования производительности.

3. Ресурсы
Оборудование : Компьютеры с разными операционными системами (Windows, macOS, Linux), мобильные устройства (Android, iOS).
Инструменты : Selenium (для автоматизации), Postman (для API-тестирования), Lighthouse (для анализа производительности).

4. Критерии завершения
Все тест-кейсы выполнены.
Все найденные дефекты задокументированы и исправлены.
Производительность и безопасность соответствуют требованиям.
Процент успешно выполненных тест-кейсов ≥ 95%.
Все критические дефекты исправлены.
Время загрузки страницы ≤ 3 секунд на всех устройствах.

1. Проверка работы поиска
ID тест-кейса : TC-01
Заголовок : Проверка функциональности поиска
Предусловия : Пользователь находится на главной странице сайта.
Шаги:
  1. Ввести в строку поиска слово "InfoSphere".
  2. Нажать кнопку "Поиск".
  3. Ввести пустой запрос и нажать "Поиск".
  4. Ввести запрос с ошибкой (например, "InfoSphre") и нажать "Поиск".
Ожидаемый результат:
  - Для корректного запроса: На странице результатов отображаются статьи, связанные с запросом "InfoSphere", время отклика ≤ 2 секунды.
  - Для корректного запроса: На странице результатов отображаются статьи, связанные с запросом "InfoSphere".
  - Для пустого запроса: Отображается сообщение об ошибке или список популярных статей.
  - Для запроса с ошибкой: Отображается сообщение "Ничего не найдено" или список рекомендаций.
Фактический результат: [Заполняется после выполнения теста].
